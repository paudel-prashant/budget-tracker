# `lib/` layout

| Folder | Purpose |
|--------|---------|
| `auth/` | API route auth helpers |
| `config/` | App config, env, navigation, layout constants |
| `db/` | Prisma client |
| `data/` | Server data loaders (dashboard, insights, budgets, reports, import) |
| `domain/` | Business logic, calculations, filters, processors |
| `services/` | CSV, serialization, PDF export |
| `validation/` | Request/body validators |
| `utils/` | Formatting, API helpers, cache revalidation |
| `theme/` | MUI theme, form styles, chart tokens |
| `types/` | Shared TypeScript types |
