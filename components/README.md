# `components/` layout

| Folder | Purpose |
|--------|---------|
| `shared/ui/` | Reusable UI primitives (cards, headers, forms, skeletons) |
| `shared/layout/` | App shell, sidebar, nav, auth layout |
| `shared/providers/` | Theme, session, snackbar |
| `dashboard/` | Home dashboard widgets and charts |
| `transactions/` | Transaction list, forms, filters |
| `budget/` | Budget management |
| `recurring/` | Recurring transactions |
| `insights/` | Financial insights page |
| `reports/` | Monthly reports |
| `settings/` | Settings and import/export |
